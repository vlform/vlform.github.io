vlsetMax

vlsetMax method set maxmium value of your Input. return false if value length is greater than the parameter.It has two parameter one is maximum value and another is error message.Fox Example if you want user to input their Name in seven letter but the user typed eight letter.When user submit the form and user get the error message.

      
      vlsetMax(7,"error message");

vlToolTip

vlToolTip method is additional method which helps you to show the tooltip when you hover over any elements.You can use it not only in the form validation or may be you can use it in form form giving a clue when user hover over the input section.
For using this tooltip you need to add css which can be found in download files inside the css place the tooltip css file in yout html. It's easy to use just with one line of js code and html you have to setup tooltip attribue let's try 

      
  $('.text').vlToolTip({
  position:"left"
});

In above Example you can see the JQuery code wich select the elements called text and vlToolTip is a method, and you can change the default position using postion which only change to top,left and bottom. Now let's workk thro our html code.

      
  <p class="text" toolTip="We can see the magic of the toolTip">
  

  Hover me to see the magic of tooltip</p>

In above example you can see the p tag which has the class of text and the attribue of tooltip and as we know now we need tooltip attribue to display the message or we can call it tooltip Now,it's time to see the output.

      
Hover me to see the tooltip
