Installation

Compatibility Note

VlForm.js This plugin need JQuery. The browser support is IE9 & + . our own customized error message. if error is not showing make sure you have passed the callback function argument in new instance of Input object. The error element class is => form-error

Direct <script> Include

Simply download and include with a script tag. VlForm.js will be registered as a global variable.

Introduction

What is VlForm.js ?

VlForm.js is the JQueryValidation Plugin provides drop-in validation for your existing forms, while making all kinds of customizations to fit your application really easy.

Getting Started

Let's Create a simple HTML form which hold username and the button to submit the form.
NOTE: Don't forget to include vlform.min.js in your HTML 

  
      HTML
      
<form>

      
 <label for="userName">Username</label>

      
<input type="text" id="username" value="" >

      
<button id="submit" >Check the validation </button>

       
</form>

Now, let's create a javascript files and include in our HTML 

  
      JS
    
 $(function(){

    
$('#submit').on('click',function(v){

    
var name = new Valid($('#username'),v);

    
name.vlRequired("Please enter the username");

  
});

   
      
});

- pass a argument v or whatever in submiting form callback function for cancel the default action if there is any error

-Create the new instance of Valid object it takes 2 arguments 1st input and 2nd e argument
